Constantes JavaScript

Palavra reservada: const

Regras:
  • não usar palavras reservadas;
  • usar nomes significativos;
  • começar com letra minúscula (camelCase);
  • sem espaços ou traços.
  • deve ser inicializado
Tipagem dinâmica
const number = 5 // number
const number = '5' // string
typeof(number) // checagem

Diferença: nas operações

  • number + number = soma
  • string + string/number = concatenação

    • Possibilidade: conversão com parseOf()

    const number = '5';
    let quadrado;
    quadrado = parseOf(number)^2;
    console.log(quadrado);
